Utopia FQA
  • Q1 : 게임의 글자가 깨져서 나와요.
    A1 : 연결 프로그램을 인터넷 익스플로어로 설정한 경우 게임의 글자가 깨지는 경우가 있습니다.
    크롬 / 네이버 웨일 / 마이크로 엣지로 연결하시면 해결됩니다.


  • Q2 : 게임을 완료하면 어떻게 하나요?
    A2 : 게임을 다시 즐기고 싶으시면 웹페이지로 다시 돌아가셔서 플레이 하시면 됩니다.


  • Q3 : 이 게임을 다른 곳에 공유해도 되나요?
    A3 : 아직 이 게임은 비공개 알파테스트 이므로 다른 곳으로 공유하지 마시기 바랍니다.


  • Q4 : 게임 도중 불편한 느낌이 듭니다.
    A4 : 게임을 즉시 중단하시고 편안한 곳에서 휴식을 취하시기 바랍니다. 만약 불편한 증세가 계속되면 의사와 상담하십시오.


  • Q5 : 궁금한 점이 있어요/게임 내에 오류가 있어요 / 개발자 에게 질문하기
    A5 : 알파테스트의 경우 개발자에게 직접 갠톡을 주시면 조치해드리겠습니다.
